# Get Started

## Requirements

* [node](https://nodejs.org)
* [yarn](https://yarnpkg.com/en/docs/install#mac-stable)

## Installation

You can use either **npm**

```bash
npm i @gfazioli/react-animatecss
```

or **yarn**

```bash
yarn add @gfazioli/react-animatecss
```
