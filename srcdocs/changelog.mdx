# V0.4.0

## New features added

* Added support for `as` property
* Added new `block` property
* Added new `repeat` property
* Added (this) Changelog file
* Added a "complex" example in `Bounce` component

## Improvements

* Improved the documentation with more examples and some fixes
* Reviewed the code under the hood
* Updates packages in particular the styled-components to v5.0

## Deprecated

* Deprecated `infinity` property. Use new `repeat` property instead
