  </h1>
</Playground>

## Repeat

You may use `repeat` to repeat the animation _n_ times. Use `infinite` for infinite loop.

<Playground>
  <h1>Say Hello to React
    <Flip animate={true} repeat="3">
      AnimateCSS
    </Flip>  
  </h1>
</Playground>

<Playground>
  <h1>Say Hello to React
    <Flip animate={true} repeat="infinite">
      AnimateCSS
    </Flip>  
  </h1>
</Playground>
