## Block

You may force the HTML element to be displayed in `block` instead of `inline-block`

<Playground>
  <h1>Say Hello to React
    <LightSpeed animate={true} block>
      AnimateCSS
    </LightSpeed>  
  </h1>
</Playground>

## Mode

<Playground>

## Repeat

You may use `repeat` to repeat the animation _n_ times. Use `infinite` for infinite loop.

<Playground>
  <h1>Say Hello to React
    <LightSpeed animate={true} repeat="3">
      AnimateCSS
    </LightSpeed>  
  </h1>
</Playground>

<Playground>
  <h1>Say Hello to React
    <LightSpeed animate={true} repeat="infinite">
      AnimateCSS
    </LightSpeed>  
  </h1>
</Playground>
