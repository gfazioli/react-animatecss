## Repeat

You may use `repeat` to repeat the animation _n_ times. Use `infinite` for infinite loop.

<Playground>
  <h1>Say Hello to React
    <Zoom animate={true} repeat="3">
      AnimateCSS
    </Zoom>  
  </h1>
</Playground>

<Playground>
  <h1>Say Hello to React
    <Zoom animate={true} repeat="infinite">
      AnimateCSS
    </Zoom>  
  </h1>
</Playground>
