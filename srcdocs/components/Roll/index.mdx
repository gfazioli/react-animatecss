## Block

You may force the HTML element to be displayed in `block` instead of `inline-block`

<Playground>
  <h1>Say Hello to React
    <Roll animate={true} block>
      AnimateCSS
    </Roll>  
  </h1>
</Playground>

## Repeat

You may use `repeat` to repeat the animation _n_ times. Use `infinite` for infinite loop.

<Playground>
  <h1>Say Hello to React
    <Roll animate={true} repeat="3">
      AnimateCSS
    </Roll>  
  </h1>
</Playground>

<Playground>
  <h1>Say Hello to React
    <Roll animate={true} repeat="infinite">
      AnimateCSS
    </Roll>  
  </h1>
</Playground>
