## Block

You may force the HTML element to be displayed in `block` instead of `inline-block`

<Playground>
  <h1>Say Hello to React
    <Hinge animate={true} block>
      AnimateCSS
    </Hinge>  
  </h1>
</Playground>

## Delay

<Playground>
  <h1>Say Hello to React
    <Hinge animate={true} delay="5s">
      AnimateCSS
    </Hinge>  
  </h1>
</Playground>

## Duration

<Playground>
  <h1>Say Hello to React
    <Hinge animate={true} duration="5s">
      AnimateCSS
    </Hinge>  
  </h1>
</Playground>

## Repeat

You may use `repeat` to repeat the animation _n_ times. Use `infinite` for infinite loop.

<Playground>
  <h1>Say Hello to React
    <Hinge animate={true} repeat="3">
      AnimateCSS
    </Hinge>  
  </h1>
</Playground>

<Playground>
  <h1>Say Hello to React
    <Hinge animate={true} repeat="infinite">
      AnimateCSS
    </Hinge>  
  </h1>
</Playground>
