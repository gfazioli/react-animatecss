Get Started

Basic usage

Say Hello to React AnimateCSS

<h1>
  Say Hello to React
  <Swing animate={true}>AnimateCSS</Swing>
</h1>

Block

You may force the HTML element to be displayed in block instead of inline-block

Say Hello to React AnimateCSS

<h1>
  Say Hello to React
  <Swing animate={true} block>
    AnimateCSS
  </Swing>
</h1>

Delay

Say Hello to React AnimateCSS

<h1>
  Say Hello to React
  <Swing animate={true} delay="5s">
    AnimateCSS
  </Swing>
</h1>

Duration

Say Hello to React AnimateCSS

<h1>
  Say Hello to React
  <Swing animate={true} duration="5s">
    AnimateCSS
  </Swing>
</h1>

Repeat

You may use repeat to repeat the animation n times. Use infinite for infinite loop.

Say Hello to React AnimateCSS

<h1>
  Say Hello to React
  <Swing animate={true} repeat="3">
    AnimateCSS
  </Swing>
</h1>

Say Hello to React AnimateCSS

<h1>
  Say Hello to React
  <Swing animate={true} repeat="infinite">
    AnimateCSS
  </Swing>
</h1>

Agnostic tag

Say Hello to React AnimateCSS

<h1>
  Say Hello to React
  <Swing animate={true} as="em">
    AnimateCSS
  </Swing>
</h1>
